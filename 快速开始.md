# 🚀 快速开始指南

## ⚠️ 重要：首先需要安装 Node.js

你的电脑上还没有安装 Node.js，这是运行这个网站的必要工具。

---

## 📥 第一步：安装 Node.js

### 方法一：官网下载（推荐）

1. **打开浏览器**，访问：https://nodejs.org/
2. **下载 LTS 版本**（左侧绿色按钮，推荐版本）
   - 文件名类似：`node-v20.x.x.pkg`
3. **双击下载的文件**，按照安装向导完成安装
4. **重启终端**（关闭并重新打开 Terminal）

### 方法二：使用 Homebrew（如果你已安装）

在终端中输入：
```bash
brew install node
```

---

## ✅ 第二步：验证安装

打开终端（Terminal），输入：
```bash
node --version
```

如果显示版本号（如 `v20.11.0`），说明安装成功！✅

---

## 🛠️ 第三步：安装项目依赖

1. **打开终端**（Terminal）
2. **进入项目文件夹**：
   ```bash
   cd /Users/royshen/hydrateddog
   ```
3. **安装依赖**（这会下载所有需要的代码库）：
   ```bash
   npm install
   ```
   - ⏱️ 等待 1-3 分钟
   - ✅ 看到 "added XXX packages" 就成功了

---

## 🎮 第四步：启动开发服务器

在终端中输入：
```bash
npm run dev
```

**等待几秒钟**，你会看到：
```
  ▲ Next.js 14.2.5
  - Local:        http://localhost:3000
  - Ready in 2.3s
```

看到这个就说明启动成功了！✅

---

## 🌐 第五步：在浏览器中打开网站

1. **打开浏览器**（Safari、Chrome、Firefox 都可以）
2. **在地址栏输入**：`http://localhost:3000`
3. **按回车键**

🎉 你应该能看到你的网站了！

---

## 🧪 第六步：测试功能

### 测试饮水记录：
1. 点击 "添加饮水记录" 按钮
2. 输入 `100`（表示 100ml）
3. 点击 "确认"
4. 看 "总饮水量" 是否变成 100ml

### 测试称重记录：
1. 点击 "添加称重记录" 按钮
2. 出门前体重输入 `10.5`
3. 回来后体重输入 `10.3`
4. 点击 "确认"
5. 看 "总尿量" 是否变成 200ml（因为 (10.5 - 10.3) × 1000 = 200）

### 测试删除：
- 点击记录旁边的垃圾桶图标 🗑️
- 确认删除
- 记录应该消失了

---

## 🛑 停止服务器

在终端中按 `Control + C`（Mac 上就是 Ctrl + C）

---

## 🌍 发布到互联网（使用 Vercel）

### 1. 注册 Vercel 账号
- 访问：https://vercel.com/
- 点击 "Sign Up"
- 使用 GitHub 账号登录（如果没有，先注册 GitHub）

### 2. 安装 Vercel CLI
在终端中输入：
```bash
npm install -g vercel
```

### 3. 登录 Vercel
```bash
vercel login
```
会打开浏览器，确认登录即可。

### 4. 部署项目
在项目文件夹中：
```bash
vercel
```

按照提示：
- 是否部署？输入 `Y`
- 是否链接现有项目？输入 `N`（第一次）
- 项目名称：直接回车
- 目录：直接回车

等待部署完成...

### 5. 访问你的网站
部署完成后，Vercel 会给你一个网址，类似：
`https://your-project-name.vercel.app`

复制这个网址，在浏览器打开，你的网站就上线了！🎉

## ❓ 遇到问题？

### 问题：`node: command not found`
**解决**：需要先安装 Node.js（见第一步）

### 问题：端口 3000 被占用
**解决**：使用其他端口
```bash
npm run dev -- -p 3001
```
然后访问 `http://localhost:3001`

### 问题：`npm install` 很慢或失败
**解决**：使用国内镜像
```bash
npm install --registry=https://registry.npmmirror.com
```

### 问题：修改代码后页面没变化
**解决**：
- 检查终端是否有错误
- 刷新浏览器（Command + R）
- 重启服务器（Control + C，然后重新 `npm run dev`）
